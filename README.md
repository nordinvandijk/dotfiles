# dotfiles
My configuration dotfiles

## Installation

### macOS

```bash
bash installers/macOS.sh
```

### Ubuntu

```bash
mkdir -p ~/Documents/GitHub
```

```bash
cd ~/Documents/GitHub
```

```bash
git clone https://github.com/nordinvandijk/dotfiles.git
```

```bash
cd dotfiles
```

```bash
source installers/Ubuntu.sh
```
