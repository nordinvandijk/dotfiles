# dotfiles

My configuration dotfiles

## Installation

### Applications

For windows

```sh
winget import Winget.json
```

### Configurations

```sh
chezmoi init --apply https://github.com/nordinvandijk/dotfiles.git
```

## Application Overview

| Type        | Solution   |
|-------------|------------|
| Terminal    | Wezterm    |
| Shell       | Nushell    |
| Prompt      | Starship   |
| Code Editor | Neovim     |
| Git Editor  | Lazygit    | 
| Containers  | Docker     |
