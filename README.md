# dotfiles
My configuration dotfiles

## Installation

### macOS

```bash
bash installers/macOS.sh
```

### Ubuntu

```bash
sudo apt update
```

```bash
sudo apt install gh
```

```bash
mkdir -p ~/Documents/GitHub
```

```bash
cd ~/Documents/GitHub
```

```bash
gh repo clone nordinvandijk/dotfiles
```

```bash
cd dotfiles
```

```bash
bash installers/Ubuntu.sh
```
